Содержит:
* Тренировка спряжений греческих глаголов (два залога и 4 основных времени). Иррегулярные, нетипичные + набор глаголов для всех паттернов.
* Тренировка склонений греческих существительных.
* Поиск спряжений для глаголов.

Установка:
* Скачать zip файл - Code кнопка справа вверху, download zip file.
* Распаковать и открыть greek.html в браузере.

Состояние тренировки сохраняется автоматически, при следующей загрузке страницы оно восстанавливается.
